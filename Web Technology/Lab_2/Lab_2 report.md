# Lab 2 Report

## Title
HTML Multimedia Integration and Web Navigation

## Objective
1. To understand and implement URL redirects and page navigation in HTML.
2. To embed and display audio files in web pages using HTML5 `<audio>` tag.
3. To embed and display video files in web pages using HTML5 `<video>` tag.
4. To integrate third-party video content using `<iframe>` (YouTube embeds).
5. To create structured web pages with semantic HTML and multimedia elements.

## Theory
Modern web applications require multimedia content to engage users. This lab explores HTML5 multimedia capabilities:

**Page Redirection**: HTTP-equiv meta tags can redirect users automatically to another URL. This is useful for forwarding obsolete pages to new locations.

**Audio Element**: The `<audio>` tag supports multiple formats (wav, mp3, ogg) with fallback options. The `controls` attribute provides play, pause, and volume controls.

**Video Element**: The `<video>` tag plays video files with native browser controls. Multiple sources ensure browser compatibility. Attributes like `width`, `height`, and `controls` manage playback behavior.

**YouTube Integration**: Using `<iframe>` with YouTube's embed code allows seamless integration of YouTube videos while maintaining responsive design.

**Semantic Structure**: Proper use of `<header>`, `<nav>`, `<article>`, `<section>`, and `<footer>` tags creates accessible, well-organized web content.
