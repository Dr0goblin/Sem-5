# Lab 5 Report

## Title
Advanced JavaScript: Arrow Functions, Recursion, Timers, Browser Storage, and Form Validation

## Objective
1. To implement and use arrow functions (`=>`) for cleaner, more concise code syntax.
2. To understand and implement recursive functions (factorial calculation).
3. To implement countdown timers using `setInterval()` and time manipulation.
4. To use browser cookies for persistent client-side data storage.
5. To implement theme toggling (dark/light mode) with CSS and JavaScript.
6. To create comprehensive form validation with multiple validation rules.
7. To provide real-time error feedback to users during form input.

## Theory
Advanced JavaScript techniques enable more sophisticated and user-friendly web applications:

**Arrow Functions**: Introduced in ES6, arrow functions provide concise syntax and lexical `this` binding. They simplify callback functions and reduce code verbosity.

**Recursion**: Functions that call themselves to solve problems that have recursive structure, such as factorial (n! = n × (n-1)!). Recursion requires a base case to prevent infinite loops.

**Timers**: `setInterval()` executes code repeatedly at specified intervals (milliseconds). `clearInterval()` stops the timer. These are used for countdowns, animations, and scheduled tasks.

**Browser Storage**: Cookies store small amounts of data on the client side, persisting across page reloads. Functions can set, get, and manage cookies using JavaScript.

**CSS Classes and toggles**: JavaScript can add/remove CSS classes to elements, enabling theme switching and dynamic styling changes without page reload.

**Form Validation**: Comprehensive validation checks multiple rules:
  - Length requirements (minimum/maximum characters)
  - Character restrictions (no numbers in names)
  - Pattern matching (email format with regex)
  - Complexity requirements (uppercase, lowercase, numbers, special characters)
