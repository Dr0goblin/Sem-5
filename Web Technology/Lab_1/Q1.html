HTML Basics

Use of <p> tag

The <p> tag in HTML is used to define a paragraph. Paragraphs are block-level elements that automatically add some margin before and after the element. This is the most commonly used HTML element for text content.

Use of <pre> tag

The <pre> tag defines preformatted text. Text within a <pre> element is displayed in a fixed-width font, and it preserves both spaces and line breaks exactly as written in the HTML source code.

This is text inside a <pre> tag.
   Notice how the spaces   are preserved.
   Also notice how line breaks
   are maintained exactly as written.

This is useful for:
  1. Displaying code examples
  2. Showing formatted text
  3. Presenting ASCII art
