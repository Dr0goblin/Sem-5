## Theory
HTML (HyperText Markup Language) is the fundamental building block of web pages. This lab covers essential HTML concepts:

**Basic Tags**: The `<p>` tag defines paragraphs with automatic spacing, while the `<pre>` tag preserves whitespace and line breaks for displaying formatted text or code.

**Tables**: HTML tables use `<table>`, `<tr>`, `<th>`, and `<td>` elements. Advanced features include `colspan` (merge columns), `rowspan` (merge rows), `<thead>`, `<tbody>`, and `<tfoot>` for semantic structure.

**Forms**: HTML forms capture user input using various elements: `<input type="text">` for text, `<input type="email">` for emails, `<input type="password">` for passwords, `<input type="radio">` for single selection, and `<input type="checkbox">` for multiple selections.

**Images and Galleries**: The `<img>` tag displays images, while `<figure>` and `<figcaption>` provide semantic wrapping. CSS flexbox can arrange images in galleries.

**Semantic HTML**: Using meaningful tags like `<header>`, `<nav>`, `<main>`, and `<footer>` improves accessibility and SEO.
