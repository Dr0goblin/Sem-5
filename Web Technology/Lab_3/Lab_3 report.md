## Theory
CSS (Cascading Style Sheets) provides powerful tools for styling and layout:

**Selectors and Properties**: CSS selectors target HTML elements. Common properties include `color`, `background-color`, `padding`, `margin`, `border`, and `font-family` for visual styling.

**Flexbox Layout**: The `display: flex` property creates flexible, responsive layouts. Properties like `flex`, `gap`, and `align-items` control element distribution and spacing.

**Div-Based Tables**: Modern web design often replaces HTML tables with div elements styled with CSS, making layouts more flexible and responsive.

**Color Coding**: Status indicators use different colors (green for complete, orange for progress, red for pending) to provide visual feedback.

**Form Styling**: Input fields, buttons, and labels can be extensively styled with CSS to create professional-looking forms with custom colors, borders, and padding.

**Responsive Design**: CSS enables responsive layouts that adapt to different screen sizes using flexible containers and relative sizing.
