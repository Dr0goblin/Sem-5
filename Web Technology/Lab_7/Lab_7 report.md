# Lab 7 Report

## Title
CRUD Operations, SQL Reports, JOIN, and OOP in PHP

## Objective
1. To implement CRUD operations on student records using PHP and MySQL.
2. To generate reports such as highest marks, lowest marks, total students, average marks, and pass/fail count.
3. To demonstrate SQL JOIN using student and parent detail tables.
4. To demonstrate OOP concepts in PHP: constructor, destructor, inheritance, polymorphism, access modifiers, getter/setter, and static members.

## Theory
This lab combines database operations and object-oriented programming in PHP.
CRUD is used to manage student data. SQL clauses such as ORDER BY, COUNT, AVG, and GROUP BY are used to generate reports. INNER JOIN is used to combine student data with parent details for failed students. OOP concepts are shown through separate examples: constructor/destructor for object lifecycle, inheritance and polymorphism for code reuse and method overriding, access modifiers for data control, getters/setters for safe access, and static members for class-level behavior.

## Codes
Main files used in this lab:
1. index.php - CRUD operations and reports
2. db.php - database connection and table creation
3. setup.sql - schema setup
4. failed_parents.php - JOIN demonstration
5. oop/constructor_destructor.php - constructor and destructor demo
6. oop/inheritance_polymorphism.php - inheritance, polymorphism, access modifiers, getter/setter demo
7. oop/static_members.php - static members demo


## Conclusion
The lab was completed successfully by integrating PHP, MySQL, and OOP concepts in one project. CRUD operations were implemented, report queries were generated, JOIN was demonstrated for failed students and parent contact details, and key OOP principles were shown with working examples.
