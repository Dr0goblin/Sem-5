## Title
JavaScript Fundamentals: Arrays, Loops, and Interactive User Input

## Objective
1. To understand JavaScript arrays and array indexing.
2. To implement loops (for loops, forEach) for iterating over arrays and groups of data.
3. To use JavaScript objects to store related data (student information).
4. To implement user interaction through prompts and button clicks.
5. To dynamically generate and display output in web pages using DOM manipulation.
6. To demonstrate mathematical operations in JavaScript (multiplication, Fibonacci sequence).

## Theory
JavaScript is a client-side scripting language that adds interactivity to web pages. This lab covers fundamental JavaScript concepts:

**Arrays**: Arrays store multiple values in a single variable. Elements are accessed using index notation (array[index]).

**Objects**: JavaScript objects group related data and properties. For example, a student object contains name, roll number, department, and project information.

**Loops**: For loops iterate a specific number of times, while forEach loops iterate over array elements. Loops are essential for processing collections of data.

**User Input**: The `prompt()` function displays a dialog box for user input. The `confirm()` function asks yes/no questions, enabling decision-making in code.

**DOM Manipulation**: `document.getElementById()` and `.innerHTML` allow dynamic page content modification based on user interaction and data processing.

**Mathematical Operations**: JavaScript supports arithmetic operations for calculations like Fibonacci sequences and multiplication tables.
